# StrawberryOS
Bare Metal Operating System for Raspberry Pi 4B

# Features
- Mini UART support
- Exception level handling

# Install
1) Run ```make```
2) Copy the following files onto sd card for Raspberry Pi
- ```config.txt```
- ```armstub8.bin```
- ```build/kernel8.img```
